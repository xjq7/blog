Nest 9.x

## Nest cli

### new

创建新项目

```sh
nest new [projectName]
```

### generate

模板生成

```sh
# 生成 module 模板代码
nest generate module [moduleName]

# 生成 controller 代码
nest generate controller [controllerName]

# 生成 service 代码
nest generate service [serviceName]

# 生成整个模块代码
nest generate resource [resourceName]
```
